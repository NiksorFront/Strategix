# Strategix

## Сервис по построению репутации для выходна на международные рынки



# Запуск Strategix (dev)

## Требуется установить 
- [Git](https://git-scm.com/install/)
- [Node.js](https://nodejs.org/en/download)

---

## macOS

### Установка Git
1) Откройте **Terminal**
2) Выполните установку homebrew :
   `/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`
3) С помощью скачанного homebrew установите git командой `brew install git`
4) Git установлен(можно проверить командой git --version), можно закрыть терминал и перейти к установке NodeJs

### Установка NodeJs
1) Перейдите на [https://nodejs.org/en/download](https://nodejs.org/en/download) и нажмтие кнопку скачивани .pkg файла для MacOs, что показана, на ![скриншоте](<Снимок экрана 2025-12-17 в 08.21.06.png>)
2) Пройдите процесс установки NodeJs
3) NodeJs установлен(можно проверить командой node -v), перейдем к сайта и CMS

### Запуск сайта с CMS
1) Скачайте файл Run-Strategix-MacOS.command
2) Переместите в папку где планируете хранить сайт
3) Зажмите «command» и кликните правой кнопкой мыши по папке
4) В всплывающем меню нажмите на пункт «Новый терминал по адресу папки»
5) В открывшемся терминале введите команду `chmod +x Run-Strategix-MacOS.command` и нажмите Return(Enter) на клавиатуре
6) Выполнив команду запустите сайт, введя команду `./Run-Strategix-MacOS.command` и нажав Return(Enter)(если с первого раза не срабатывает попробуйте ввести её вновь)
7) После зрагрузки сайт можно открыть в любом браузере по [http://localhost:3000/](http://localhost:3000/)
8) CMS система сайта доступа по ссылке [http://localhost:3000/CMS](http://localhost:3000/)
